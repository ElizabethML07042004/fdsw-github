Mis Proyectos

App Huellitas

Huellitas es una aplicación innovadora diseñada para facilitar el proceso de adopción de mascotas. Permite a los usuarios explorar perfiles de mascotas disponibles para adopción, registrar animales perdidos, y acceder a información útil sobre el cuidado de mascotas. La aplicación incluye características para gestionar una lista de favoritos, así como un formulario para registrar y buscar mascotas.
